# LeanbackLauncher-kotlinx — План развития и дорожная карта проекта

Этот проект представляет собой кастомный лаунчер для **Android TV** и **Amazon Fire TV**, разработанный на базе Google Leanback Launcher и находящийся в процессе миграции с Java на Kotlin.

---

## 📋 Основные цели и этапы разработки

### 🛠 Этап 1: Исправление критических багов
Устранение существующих ошибок для повышения стабильности работы лаунчера на повседневной основе.
* **[В ПРОЦЕССЕ] Баг фонового воспроизведения звука:** При выходе из IPTV/YouTube/плееров по кнопке Home звук продолжает играть, пока пользователь не нажмет кнопку навигации.
  * *Решение:* Перевод тем `LeanbackLauncherTheme` и `AppCompatFullScreenTheme` в непрозрачный режим (`windowIsTranslucent = false`).
* **Другие баги UI/UX:** Выявление и устранение сбоев анимации, падений при долгой работе, проблем с фокусом на некоторых моделях ТВ.

### 📺 Этап 3: Новые каналы рекомендаций (Channel Previews)
Расширение возможностей домашнего экрана для новых версий Android TV.
* **Многострочные рекомендации:** Переход от одной общей ленты рекомендаций к возможности отображения нескольких тематических каналов.
* **Каналы по приложениям:** 
  * Добавление возможности выбора конкретного приложения, для которого создается отдельная строка (лента) рекомендаций.
  * Интерфейс настройки: сначала пользователь выбирает приложение из списка, затем под него генерируется и закрепляется отдельная горизонтальная лента с контентом этого приложения.

---

### 💾 Этап 4: Система резервного копирования (Backup & Restore)
Защита настроек пользователя при переустановке приложения.
* **Что нужно сохранять:**
  * Порядок и группировку баннеров (приложения, избранное, игры).
  * Кастомные иконки, цвета рамок и настройки сетки.
  * Идентификаторы виджетов и общие настройки интерфейса.
* **Реализация:**
  * Экспорт настроек в файл (например, в формате JSON) во внутреннюю или внешнюю память.
  * Функция импорта для быстрого восстановления структуры рабочего стола «в один клик».
